# 📝 My To-Do App

A beautiful React To-Do List app with a dark glassmorphism design.

## 🚀 How to Run in VS Code

### Step 1 — Open the folder in VS Code
Unzip/extract this folder, then open it in VS Code.

### Step 2 — Open the Terminal
Press **Ctrl + `** (backtick) to open the terminal inside VS Code.

### Step 3 — Install dependencies
```bash
npm install
```
This will download all required packages (takes ~1-2 minutes the first time).

### Step 4 — Start the app
```bash
npm start
```
The app will automatically open at **http://localhost:3000** in your browser! 🎉

---

## ✨ Features
- ✅ Add tasks (type + Enter or click +)
- ✔️ Check/uncheck tasks
- ✏️ Edit tasks inline
- 🗑️ Delete tasks
- 🔍 Filter: All / Active / Completed
- 📊 Progress bar
- 🧹 Clear all completed tasks
